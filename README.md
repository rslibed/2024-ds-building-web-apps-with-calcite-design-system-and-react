## Steps to run web app locally, in terminal, run:

1. `git clone https://github.com/rslibed/2024-ds-building-web-apps-with-calcite-design-system-and-react.git`
2. `cd 2024-ds-building-web-apps-with-calcite-design-system-and-react`
3. `yarn install`
4. `yarn start`

Note: yarn and git is required run the above commands. 

Browser will automatically open at `http://localhost:3000`.
